## Setup Instructions

Note: Please ensure you have installed <code><a href="https://nodejs.org/en/download/">nodejs</a></code>

To preview and run the project on your device:

1) Open project folder in <a href="https://code.visualstudio.com/download">Visual Studio Code</a>
2) In the terminal, run `npm install`
3) Run `npm start` to view project in browser

To view this project locally:

1. Navigate to the project directory:
   cd Aroc

2. Open `build/index-cockpit.html` in your browser:

   Open the `build/index-cockpit.html` file in your preferred browser.

## Assets

All project assets are stored in the `assets/` directory. Images are specifically located in `../static/assets/images/`.
